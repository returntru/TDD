# Test-Driven Development
This is a source code of examples in *Test-Driven Development* by Kent Beck.

### Built with
* [Maven](https://maven.apache.org/) - Dependency Management
* [JUnit](https://junit.org/junit4/) - Simple framework to write repeatable tests

## Instructions

**Install Java**

Download and install Java Development Kit from this [link](https://www.oracle.com/java/technologies/javase/javase-jdk8-downloads.html).

**Verify**

To verify that Java is installed successfully, you need to run these commands:

`java -version`

`javac -version`

**Starting up**

I personally used **Intellij IDEA** to run it. You have the download link [here](https://www.jetbrains.com/idea/download/)
